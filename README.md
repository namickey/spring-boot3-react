# spring-boot3-react

## 構成

- フロントエンド
    - react

- バックエンド
    - spring-boot

## 環境

- nodejs v24
- vite
- jdk 21

## spring-boot and frontend

```
コマンドプロンプトで実行
mvnw.cmd spring-boot:run

http://localhost:8080/
```



## frontend only
```
npm create vite@latest
```

```
インストール
npm install

開発サーバ起動
npm run dev
```

```
本番向けビルド
npm run build

プレビュー
npm run preview
```
